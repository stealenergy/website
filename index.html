 Seguidores de Linha 

Conceito

Um robô seguidor de linha é um dos conceitos básicos para quem esta começando na área da robótica, por ser um robô totalmente autônomo, programado apenas para ler a linha escura de uma pista, através de sensores de luz.

Como Funciona

Ele funciona basicamente com uma placa, como um Arduino ou Raspberry Pi, por exemplo, uma fonte de corrente elétrica, alguns componentes como rodas, motores, sensores..., E a programação dele!

Autodesk Tinkercad

O Tinkercad é um programa da Autodesk de modelagem 3D online gratuito que roda em um navegador da web.

Nele podemos também projetar e fazer a simulação de circuitos em Arduino, como no exemplo abaixo, em que projetamos nosso robô.

Código do Robô

Programamos o robô com o objetivo de manter sempre os dois sensores sobre a linha.

Na explicação abaixo, detalhamos o funcionamento do nosso primeiro código.

Nesta primeira parte, são definidos os nomes para as portas que serão usadas para a execução deste código.

Este não é um processo obrigatório, porém facilita muito o desenvolvimento do programa.


// MOTOR
#define IN1 2
#define IN2 4
#define IN3 7
#define IN4 5
#define ENA 3
#define ENB 6

// LED RGB
#define LEDR 9
#define LEDB 10
#define LEDG 11

// SENSOR
#define S1 A0
#define S2 A1
#define S3 A2
#define S4 A3

Nesta fase, através das comandas "pinMode" e "analogWrite".

São setadas as funções de cada uma das portas.


void setup()
{
  // MOTOR
  pinMode(IN1,OUTPUT);
  pinMode(IN2,OUTPUT);
  pinMode(IN3,OUTPUT);
  pinMode(IN4,OUTPUT);
  pinMode(ENA,OUTPUT);
  pinMode(ENB,OUTPUT);
  analogWrite(ENA,255);
  analogWrite(ENB,255);
  
  // LED RGB
  pinMode(LEDR, OUTPUT);
  pinMode(LEDB, OUTPUT);
  pinMode(LEDG, OUTPUT);
  
  Serial.begin(9600);
}

Nesse processo de loop, são criadas as variaveis para as quatros sensores.

Estas variaveis são inseridas em umas estrutura condicional que segue a seguinte condição:...


void loop()
{
  // Leitura dos sensores
  int s1, s2, s3, s4;
  
  s1=analogRead(S1);
  s2=analogRead(S2);
  s3=analogRead(S3);
  s4=analogRead(S4);
  
  int luz = 500;
  
  Serial.print(s1);
  Serial.print(" ");
  Serial.print(s2);
  Serial.print(" ");
  Serial.print(s3);
  Serial.print(" ");
  Serial.print(s4);
  Serial.print(" ");

...Partimos do principio que a linha indicadora da direção é na cor clara (branco), e seu envolto é de cor escura (preto).

Se os sensores localizados no lado esquerdo e direito não captarem a luminosidade da linha (Na cor branca), e apenas as sensores centrais tiverem estes valores, o cogido interpretará que ambos motores devem ir na mesma velocidade e para frente, com o LED indicando a cor verde.

if(s1<luz && s2>luz && s3>luz && s4<luz){
  // FRENTE
  Serial.println(" Frente ");
  
  digitalWrite(LEDR, LOW);
  digitalWrite(LEDG, HIGH);
  digitalWrite(LEDB, LOW);
  
  digitalWrite(IN1,HIGH);
  digitalWrite(IN2,LOW);
  digitalWrite(IN3,LOW);
  digitalWrite(IN4,HIGH);
Se, eventualmente, o sensor da lado esquerdo não captar luminosidade, mas os demais sim, então o carrinho irá virar para o Iado direito e acenderá o LED na cor vermelha.

}else if (s1>luz && s2<luz && s3<luz && s4<luz){
  // DIREITA
  Serial.println(" Direita ");
  
  digitalWrite(LEDR, HIGH);
  digitalWrite(LEDG, LOW);
  digitalWrite(LEDB, LOW);
  
  digitalWrite(IN1,LOW);
  digitalWrite(IN2,HIGH);
  digitalWrite(IN3,LOW);
  digitalWrite(IN4,HIGH);

Na entanto, se os sensores da direita não captarem a linha na cor branca mas os demais sim, a carrinho irá virar para o lado esquerdo, e o LED também ficará na cor vermelha


}else if (s1<luz && s2<luz && s3<luz && s4>luz){
  // ESQUERDA
  Serial.println(" Esquerda ");
  
  digitalWrite(LEDR, HIGH);
  digitalWrite(LEDG, LOW);
  digitalWrite(LEDB, LOW);
  
  digitalWrite(IN1,HIGH);
  digitalWrite(IN2,LOW);
  digitalWrite(IN3,HIGH);
  digitalWrite(IN4,LOW);
E finalizando o código, se nenhuma das condições anteriores forem atendidas, o carinho ficará parado e o LED estará na cor azul

}else{
  // DESLIGADO
  Serial.println(" Parado ");
  
  digitalWrite(LEDR, LOW);
  digitalWrite(LEDG, LOW);
  digitalWrite(LEDB, HIGH);
  

  digitalWrite(IN1,LOW);
  digitalWrite(IN2,LOW);
  digitalWrite(IN3,LOW);
  digitalWrite(IN4,LOW);
  }
}

Componentes

Lista de componentes que usamos para montagem do seguidor de linha.

Arduino

Responsável por ler os sinais recebidos e aplicar a inteligência com base no código programado.

Bateria 9v

Responsável pela alimentação do circuito.

Motores

Responsáveis pela movimentação do carrinho, com base nos comandos recebidos pelo Arduino.

Placa de Ensaio

Placa por onde todo o circuito é conectado.

Porta Lógica L293D

Usada para controle dos motores.

Interruptor Deslizante

Responsável por ligar/desligar o circuito.

Resistências

Responsáveis em auxiliar no acionamento do LED

Sensor de Luminosidade

Responsável por fazer a leitura e identificar a diferença de cor na pista.

Led RGB

Responsável por sinalizar, através da cor, o estado atual do carrinho.

Os componentes podem variar a depender do projeto!
